# Projeto Desenvolvimento Web - Fake Pinterest

### Contexto
 Cópia do site Pinterest, usuário pode criar uma conta, 
 postar imagens e ver imagens e perfis de outros usuários.
 
### Ferramentas Usadas
Flask:
- Ferramenta Python usada para desenvolvimento WEB 
- Usada para escopo inicial do Site, para autenticação com o Flask Bycrypt, criação do Banco de Dados com Flask SQLalchemy e sistema de Login com Flask Login

HTML:
- Linguagem de marcação usada para dar forma ao site

CSS:
- Mecanismo usado para adicionar o estilo de Pinterest a página Web, aplicada
diretamente no HTML
